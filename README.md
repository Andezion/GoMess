# GoMess

Простой TCP чат-мессенджер на Go.

## Особенности

- Многопользовательский чат в реальном времени
- TCP подключения
- Graceful shutdown
- Поддержка контекста

## Запуск

### Сервер
```bash
go run cmd/server/main.go
```

Сервер запускается на порту `:8080`

### Клиент
```bash
go run cmd/client/main.go
```

## Команды

- `/quit` - выход из чата
- `/users` - список пользователей
- Любой другой текст отправляется как сообщение

## Структура

```
cmd/
  server/   - точка входа сервера
  client/   - точка входа клиента
server/     - логика сервера
```
